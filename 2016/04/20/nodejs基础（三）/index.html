nodejs基础（三）

发表于 2016-04-20


express：
http://www.expressjs.com.cn/
express脚手架：
用来快速搭建工程的工具
安装脚手架：
npm install express-generator -g
创建工程：
express 工程名 -e
安装所有组件（首先需要进到工程目录中）
npm install
启动工程
npm start
默认端口号：
3000
修改端口：
bin\www文件

中间件：
middleware，服务器接收到请求为开始，响应请求为结束，这两者之间的处理由中间件来进行。
作用：对服务器功能进行扩展。

后端模板：
数据的填充过程是在服务器端完成的，填充完成以后发送给客户端。
优点：提高SEO

前端模板：
数据的填充过程是在浏览器上完成的。
缺点：不利于SEO
