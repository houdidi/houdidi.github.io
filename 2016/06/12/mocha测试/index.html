mocha

发表于 2016-06-12


Mocha（发音”摩卡”）诞生于2011年，是现在最流行的JavaScript测试框架之一，在浏览器和Node环境都可以使用。
所谓”测试框架”，就是运行测试的工具。通过它，可以为JavaScript应用添加测试，从而保证代码的质量。

chai是断言库
所谓”断言”，就是判断源码的实际执行结果与预期结果是否一致，如果不一致就抛出一个错误。
例如：
expect(add(1, 1)).to.be.equal(2);

安装mocha
npm install -g mocha

编写测试用例
function add(x, y) {
return x + y;
}

function multi(x, y) {
return x * y;
}

module.exports = {
add: add,
multi: multi
};

var model = require(‘./add.js’);
var expect = require(‘chai’).expect;

describe(‘加法函数的测试’, function() {
// it代表一个测试用例，里面可以包含一个或多个断言expect
it(‘1 加 1 应该等于 2’, function() {
expect(model.add(1, 1)).to.be.equal(2);
});
it(‘2 乘 3 应该等于 6’, function() {
expect(model.multi(2, 3)).to.be.equal(6);
});
});

运行测试用例
mocha add.test.js(测试代码)
